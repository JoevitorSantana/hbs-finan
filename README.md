# 🙏 HBS Finan

**HBS Finan** é um sistema de gestão voltado para templos religiosos, com o objetivo de organizar e controlar de forma transparente as doações, eventos, despesas e participantes envolvidos nas atividades da instituição.

## 🧾 Visão Geral

O sistema permite o gerenciamento completo de:

- Participantes, funcionários, produtos e grupos
- Doações monetárias, alimentícias e materiais
- Estoque de itens recebidos e distribuídos
- Transações financeiras, abertura e fechamento de caixa
- Organização e gestão de eventos
- Controle de despesas e doações para outras instituições
- Relatórios detalhados com filtros (tipo, período, participante, etc.)

Com foco em **segurança e transparência**, a aplicação garante o acesso autorizado aos dados por meio de autenticação e comunicação segura via HTTPS.
